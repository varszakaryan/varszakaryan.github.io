# Event Invitation Landing Page

A beautiful, modern, and responsive landing page for event invitations.

## Features

- **Modern Design**: Clean, elegant design with gradient backgrounds and smooth animations
- **Responsive Layout**: Works perfectly on desktop, tablet, and mobile devices
- **Event Information**: Displays event name, location, date, time, and details
- **Interactive Elements**: RSVP button with user interaction
- **Smooth Animations**: Fade-in effects and hover animations for better user experience

## Files Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling and responsive design
- `script.js` - JavaScript for interactivity
- `README.md` - This file

## Customization

To customize the event details, edit `index.html` and update:

1. **Event Name**: Change the text in `<h1 class="event-name">`
2. **Event Tagline**: Update the text in `<p class="event-tagline">`
3. **Date & Time**: Modify the date and time in the first detail card
4. **Location**: Update the location information in the second detail card
5. **Event Details**: Edit the list items in the third detail card
6. **Contact Information**: Update the contact details in the footer

## Usage

Simply open `index.html` in a web browser to view the landing page. No build process or server required.

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
